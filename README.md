**Space-Dev FPS Menu**

A lightweight **FPS menu** built for Qbox.

Allows players to adjust graphics settings to improve performance, especially for low-end PCs. Simple UI and low resource usage.

**Support / Discord:**
https://discord.gg/aMZJrXy4uN

https://streamable.com/faowxe

**Features**

- FPS / performance presets

- Clean and simple UI

- Qbox compatible

- Easy to install

- Command: /fps

**Usage**

- Use /fps in-game to open the FPS menu.

**Installation**

- Add the resource to your resources folder

- Ensure it in server.cfg 

- Restart the server
